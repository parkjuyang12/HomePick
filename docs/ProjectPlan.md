# HomePick 프로젝트 기획

## 1. 프로젝트 개요

본 프로젝트는 **주기적으로 갱신되는 공공 부동산 실거래가 데이터와 민간 매물 데이터를 통합하여**, 사용자가 **지도 기반으로 최신 실거래가와 가격 변동 이력, 현재 매물 상황을 직관적으로 파악**할 수 있도록 돕는 부동산 정보 서비스입니다.

단순한 데이터 조회를 넘어,
- “지금 이 지역에서 매물의 실제 거래가는 얼마인가?”
- “이 매물은 가격이 어떻게 변해왔는가?”
- “실거래가 대비 현재 나와 있는 매물 가격은 어떤 수준인가?”

와 같은 **의사결정에 직접 도움이 되는 정보 제공**을 목표로 합니다.

---

## 2. 문제 정의 (Why)

### 2.1 기존 부동산 정보 서비스의 한계

현재 대부분의 부동산 플랫폼은 다음과 같은 문제를 가집니다.

- 실거래가 데이터는 **과거 조회용 정보**에 머무름
- 가격 변화가 “언제, 왜” 일어났는지 파악하기 어려움
- 매물 정보와 실거래가가 **분리되어 제공**
- 공공데이터의 갱신 주기 특성상:
  - 중복 데이터
  - 수정 데이터
  - 상태 변화(거래 완료 등)를 명확히 구분하기 어려움

결과적으로 사용자는
> “이 가격이 비싼 건지, 싼 건지, 최근에 변한 건지”
를 스스로 판단해야 함

---

## 3. 프로젝트 목표 (What)

### 🎯 핵심 목표

1. **가장 최근의 실거래 상태를 신뢰성 있게 제공**
2. **가격 및 상태 변화 이력을 명확한 이벤트로 제공**
3. **실거래가 + 현재 매물 정보를 한 화면에서 통합**
4. **지도 기반 탐색 + 대화형 질의(Chatbot) 지원**

---

## 4. 서비스가 제공하는 주요 기능

### 4.1 지도 기반 실거래가 시각화

- 서울/경기 지역을 중심으로 지도에 매물 마커 표시
- 각 마커는 **해당 매물의 가장 최근 실거래가**를 표현
- 지역별 실거래 흐름을 한눈에 파악 가능

---

### 4.2 매물 상세 정보 제공

지도에서 특정 매물을 클릭하면:

#### 📌 실거래가 변화 히스토리
- 가격 변동 타임라인
- 단순 스냅샷이 아닌 “변경 이력” 중심 제공

#### 📌 현재 나와 있는 매물 정보
- 민간 플랫폼에서 실시간 수집된 매물 리스트
- 실거래가 대비 현재 매물 가격 비교 가능

---

### 4.3 핫한 매물 추천 (추후 구현)

- 최근 거래 빈도
- 가격 급변 여부
- 특정 기간 내 관심도 상승 지역

등을 기반으로 **“요즘 주목할 만한 매물/지역” 추천

---

### 4.4 실거래가 추이 분석 (추후 구현)

- 지역별 / 기간별 실거래가 변화
- 가격 상승·하락 구간 시각화
- 단순 평균이 아닌 **실제 거래 이벤트 기반 분석**

---

### 4.5 부동산 정보 챗봇

- 서울/경기 지역 부동산 관련 질의 응답
- 예:
  - “최근 6개월 동안 강남구 아파트 실거래가 흐름 알려줘”
  - “이 매물은 언제 가격이 떨어졌어?”
- 실거래가 + 히스토리 + 최신 상태를 바탕으로 응답


---

## 5. 기대 효과

### 사용자 관점

- 실거래가에 대한 **신뢰도 상승**
- 가격 변화 맥락을 이해한 합리적 판단 가능
- 지도 기반 탐색 + 분석 + 대화형 질의의 결합

### 시스템 관점

- 재처리 가능한 데이터 파이프라인
- 공공데이터 특성에 강한 구조
- 확장 가능한 스트리밍 아키텍처

---

## 6. 요약

> 이 프로젝트는  
> **공공 부동산 실거래 데이터를 이벤트 스트림으로 해석하고,  
> 가격과 상태의 변화를 명확한 히스토리로 재구성하여,  
> 사용자가 ‘지금’과 ‘과거’를 동시에 이해할 수 있도록 돕는  
> 지도 기반 부동산 인텔리전스 서비스입니다.**
